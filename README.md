1. **Clone the Repository**:

   ```bash
   git clone https://github.com/muneebtech07/Tour-and-travel-landing-page-next-js.git
   cd Tour-and-travel-landing-page-next-js
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Run the Development Server**:

   ```bash
   npm run dev
   ```

   ```bash
   npm run build
   ```
